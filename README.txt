Visionary Machines — Romanticism & Art/Tech Exhibition Website
================================================================

如何查看本网站 / HOW TO VIEW
----------------------------
1. 解压本文件夹(右键 → 解压 / Extract All)。
   Unzip this folder first.
2. 打开解压后的文件夹。
   Open the unzipped folder.
3. 双击 "index.html",它会在浏览器中打开。
   Double-click "index.html" — it opens in your web browser.

重要 / IMPORTANT
----------------
所有文件必须放在同一个文件夹里,不要单独移动 index.html。
"assets" 文件夹存放了全部图片;如果把 index.html 单独拿出去,图片就会无法显示。
Keep all files together. The "assets" folder holds every image; if you move
index.html out on its own, the images and styling will break.

首次打开建议联网,以便正确加载网页字体。展览内容和图片离线也能正常显示。
An internet connection is recommended on first open so the web fonts load
correctly. The content and images work offline either way.

文件夹内容 / CONTENTS
---------------------
  index.html ........ 网站主页(打开这个 / open this)
  styles.css ........ 视觉样式 / styling
  *.js / *.jsx ...... 交互与内容数据 / interactivity & content
  assets/ ........... 全部图片 / all images
